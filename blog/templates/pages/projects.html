XKCD Almamater

On April 1st, 2013, Randall Monroe, author of the website XKCD posted a contest as part of his Aprils Fool's day comic. The contest was simple. Randall posted the output of Skein hash, and competitors were to attempt to find inputs that when hashed with the same algorithm were as close as possible to the has that Randall posted.

The original method that many people were using to generate hashes was a simple program that started hashing random text and checked the output. This program accepted an input that would change which word it started hashing at, but most people were unaware of the optin and were completing the same hashes as everyone else.

Max Peterson and I quickly threw together a web service that would hand out different "blocks" of text strings to hash and altered the program to pull from that web service.

Max and I then disseminated the client program out among Iowa State students so that they could run it on their personal desktops and a number of university resources.

As a result of all of this, Iowa State tied for 9th place among 184 schools, beating teams such as Harvard, Yale, Duke, and Iowa.

We lost to teams such as Carnegie Mellon, MIT, Columbia, and Berkely.

The source code of the web servive is hosted on Github

IScorE

IScorE is a scoring system for Iowa State's Cyber Defense Competitions which I have assisted Max Peterson in developing.

Cyber Defense Competitions are competitions where students host a number of network services and have to protect them from industry professionals that are attempting to break into their networks.

IScorE handles a huge number of portions of the competition.

  • Scanning student's services for uptime
  • All aspects of scoring for the competition
  • Wiki for attacking team to share information
  • Allows students and attacking team to communicate
  • Allows student teams to administer their own DNS
